# Snake-Game-1
"O jogo foi desenvolvido na linguagem de programação Python utilizando a biblioteca Pygame, que é uma ferramenta usada para criar jogos. O objetivo do jogo é controlar a cobra através das setas do teclado, coletando os alimentos para ganhar pontos e aumentar de tamanho. O jogo termina quando a cobra bate na parede, em si mesma ou no obstáculo que aparece na tela. Para jogar novamente, basta apertar Enter, e para sair do jogo, a tecla Esc."

Controles:

Seta para cima: sobe
Seta para baixo: desce
Seta para esquerda: vai para a esquerda
Seta para direita: vai para a direita
Enter: reinicia o jogo
Esc: fecha o jogo

Como executar:

Abrir o arquivo no Visual Studio Code.
Executar o arquivo Python.
O Pygame abrirá uma janela com o jogo pronto para jogar.

Linguagem Utilazada 
Python
IDE Utilizada Visual Studio Code
